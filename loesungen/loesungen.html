Lösungen zum Web-Tutorial

1. HTML: Die Struktur einer Webseite

HTML (HyperText Markup Language) beschreibt den Inhalt einer Webseite.

Erstelle eine einfache Webseite 

<!doctype html>
<html lang="de">
<head>
  <meta charset="utf-8">
  <title>Meine erste Webseite</title>
</head>
<body>
  <h1>Willkommen zu meiner Webseite</h1>
  <p>Dies ist ein Absatz.</p>
</body>
</html>
Erstelle eine Liste 

<ul>
  <li>Erstes Element</li>
  <li>Zweites Element</li>
  <li>Drittes Element</li>
</ul>

<ol>
  <li>Erstes Element</li>
  <li>Zweites Element</li>
  <li>Drittes Element</li>
</ol>
Bilder einfügen 

<img src="bild.jpg" alt="Beschreibung des Bildes">
Links erstellen 

<a href="https://www.beispiel.com">Externer Link</a>
<a href="#ziel">Interner Link</a>
Formulare erstellen 

<form>
  <label for="name">Name:</label>
  <input type="text" id="name" name="name">
  <br>
  <label for="nachricht">Nachricht:</label>
  <textarea id="nachricht" name="nachricht"></textarea>
  <br>
  <button type="submit">Absenden</button>
</form>

2. CSS: Das Design der Webseite

CSS (Cascading Style Sheets) verleiht deiner Webseite Stil.

Grundlegende Stile anwenden 

<style>
  p {
    color: blue;
    font-size: 16px;
  }
  h1 {
    color: red;
  }
  ul {
    background-color: lightgrey;
  }
</style>
Box-Modell verstehen 

<style>
  div {
    border: 1px solid black;
    padding: 10px;
    margin: 20px;
  }
</style>
Flexbox Layout erstellen 

<style>
  .container {
    display: flex;
    justify-content: center;
    align-items: center;
  }
  .item {
    margin: 10px;
  }
</style>

<div class="container">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
</div>
Hover-Effekte hinzufügen 

<style>
  button:hover {
    background-color: yellow;
    color: black;
  }
</style>

<button>Hover über mich!</button>
Responsive Design umsetzen 

<style>
  @media (max-width: 600px) {
    .container {
      flex-direction: column;
    }
  }
</style>

<div class="container">
  <header>Header</header>
  <main>Hauptinhalt</main>
  <footer>Footer</footer>
</div>

3. JavaScript: Interaktivität der Website

Mit JavaScript kann man Webseiten interaktiv machen.

Variablen und Datentypen 

<script>
  let zahl = 5;
  const text = "Hallo Welt";
  let wahr = true;
  console.log(zahl, text, wahr);
</script>
Einfache Funktionen 

<script>
  function summe(a, b) {
    return a + b;
  }
  console.log(summe(2, 3));
</script>
Event-Listener 

<script>
  document.querySelector("button").addEventListener("click", function() {
    console.log("Button wurde geklickt!");
  });
</script>
Arrays und Schleifen 

<script>
  let array = [1, 2, 3, 4, 5];
  for (let i = 0; i < array.length; i++) {
    console.log(array[i]);
  }
</script>
DOM-Manipulation 

<script>
  document.querySelector("button").addEventListener("click", function() {
    document.querySelector("div").textContent = "Inhalt wurde geändert!";
  });
</script>
